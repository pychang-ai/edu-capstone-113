> 113 屆專題培訓 Repo（public，學生端）
> 指導教授：張珀銀（海事AI實驗室）

## Repo 用途

學生透過 Fork + PR 繳交培訓作業和進度報告。

## 結構

| 目錄 | 用途 |
|---|---|
| `training/` | 培訓教材（5 份：Git、Linux、Python、論文閱讀、實驗室工具） |
| `members/` | 學生個人資料夾（Fork 後在此建立 `學號/` 目錄） |
| `progress/` | 月進度報告模板 |
| `resources/` | 學習資源連結 |

## 工作流

1. 學生 Fork 本 repo 到自己的 GitHub 帳號
2. 在 `members/學號/` 下完成培訓作業
3. 透過 PR 繳交回主 repo
4. PI 在 PR 上 review + 回饋
